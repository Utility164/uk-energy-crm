# UK Energy CRM

## How to run this locally
1. Install Node.js from https://nodejs.org (if not already installed)
2. Open a terminal in this folder
3. Run: npm install
4. Run: npm run dev
5. Open the link shown (usually http://localhost:5173)

## How to deploy live (Vercel - free)
1. Create a free account at https://vercel.com
2. Create a free account at https://github.com
3. Upload this folder to a new GitHub repository
4. In Vercel, click "Add New Project" → Import your GitHub repo
5. Click Deploy — Vercel auto-detects Vite and builds it
6. You'll get a live URL like https://uk-energy-crm.vercel.app
